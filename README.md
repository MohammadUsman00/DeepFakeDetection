## Deepfake Detection System (Local)

This repository will contain a complete final-year project:

**AI-Based Deepfake Detection and Media Authenticity Verification System**

### Components (planned)
- **Frontend**: React + Material UI + Recharts
- **Backend**: FastAPI + SQLite + BackgroundTasks
- **ML Inference**: PyTorch (CPU-first) + EfficientNet-B0 pretrained
- **Video pipeline**: OpenCV frame sampling (1 fps, max 150 frames)
- **Face detection**: MTCNN (largest face per frame)
- **Explainability**: Grad-CAM overlays for Top-K suspicious frames (K=5)
